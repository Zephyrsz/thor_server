# 语音对话界面（Speech-to-Speech）UI 修改说明

> 对照原型页面「Browser voice UI」→ 现版「Real-time speech AI」。
> 本文按**页面区域 / 元素**列出修改内容、目的，以及对应的**前端调整建议**，方便落到实际代码。
> 版本：含「实时交互窗口 + 吉祥物」「对话记录固定高度滚动」「左右两栏底部等高」等最新改动。

---

## 一、整体布局

| 项 | 原版 | 现版 | 前端调整建议 |
|---|---|---|---|
| 左列 | Live transcript / Speech output / Conversation 三个独立面板竖排 | 顶部**实时交互窗口区**（两个并排窗口）+ 下方**对话记录**（固定高度可滚动） | 交互窗口绑定"当前"实时状态，对话记录绑定历史消息列表 |
| 右列 | Server settings（全部参数）+ Event log | 会话控制 / 系统提示词 / 设备 / 高级设置(折叠) / 开发者日志(折叠) | 按「常用 → 次要 → 调试」分层 |
| 双栏高度 | 各自然高度 | **左列底部与右列底部对齐（等高）** | 见第六节的高度同步逻辑 |
| 配色 | 单一青色，双方无区分 | 用户 = 青 `#4fe0bf`，助手 = 蓝 `#6ea8ff` | 定义语义色 token `--user` / `--assistant` |

## 四、实时交互窗口区（本次重点，恢复并升级原窗口）

原「Listening for speech」和「Assistant audio stream」两个面板**恢复并并排放在对话记录上方**，各配一个**吉祥物形象**表示"你"与"AI"的交互，两只角色朝向中间空隙相对。

### 4.1 左窗口 · 你（Listening for speech）
- 青色主题，右上角「你 · YOU」徽标。
- 吉祥物：圆角方脸 + 眨眼(`blink`) + 整体轻浮动(`bob`) + 听觉光环脉冲(`ring`)，右下角麦克风小胶囊。
- 内容：`正在聆听 LISTENING` 状态点 + 实时转写文本 + 输入电平条形动画。

**前端建议**：把 ASR partial 文本渲染到该窗口；`connected && micOn` 时显示监听态与电平。

### 4.2 右窗口 · AI（Assistant audio stream）
- 蓝色主题，右上角「语音助手 · AI」徽标。
- 吉祥物：圆角方脸 + 眨眼 + 浮动 + 说话嘴型动效(`talk`) + 两侧耳机耳罩。
- 内容：`正在播放 SPEAKING` 状态点 + 波形动画 + 播放器（播放键 / 进度条 / 时长）。

**前端建议**：TTS 播放中驱动波形与嘴型动效；播放器绑定当前音频流。

### 4.3 吉祥物实现说明
- 全部由基础形状（圆/圆角矩形）+ CSS 关键帧动画组成，无位图/SVG 资源依赖。
- 关键帧：`bob`（浮动）、`blink`（眨眼）、`talk`（说话）、`ring`（光环扩散）、`pulseDot`（状态点）。

## 五、对话记录（固定高度 + 滚动）
- 标题「对话记录 · Conversation」，右上角图例（你/语音助手）与轮次计数。
- **固定高度**：卡片高度由布局决定（见第六节），内容超出时**内部区域垂直滚动，右侧出现滚动条**。
- 双向气泡：用户右对齐(青)、助手左对齐(蓝)；助手音频消息内含播放器与音频摘要。
- 实时"聆听/播放"状态已上移到顶部交互窗口，流内不再重复显示。

**前端建议**：滚动容器 `overflow-y:auto; min-height:0`；新消息到达时把容器 `scrollTop` 置底（不要用 `scrollIntoView`）。

---

## 六、左右两栏底部等高（高度同步逻辑）
需求：左列（交互窗口 + 对话记录）底部与右列（结尾为"开发者日志"）底部对齐。

实现方式：
- 外层为两列 `grid`（`minmax(0,1fr) 372px`，`align-items:start`）。
- 左列为 `flex` 纵向，对话记录 `flex:1; min-height:0` 占据剩余空间。
- 通过 `ResizeObserver` 监听右列高度，把**左列高度**同步为右列 `offsetHeight`；右列折叠区展开/收起时自动重算。
- 因此右列内容高度是"基准"，左列对话记录自动伸缩填满 → 两栏底部始终齐平。

**前端建议**：给左列容器 `data-role="leftcol"`、右列 `data-role="rightcol"`；在挂载与窗口 `resize`、右列尺寸变化时执行 `left.style.height = right.offsetHeight + 'px'`。

## 七、右列控制区
### 7.1 会话控制（操作按钮分层）
1. **主操作**：整宽高亮「连接服务器 / 断开连接」。
2. **次操作**：整宽描边「启动/停止麦克风」，带状态圆点。
3. **弱操作**：文字链接「清空对话历史」（建议加二次确认）。

### 7.2 系统提示词 Instructions（常驻窗口）
独立卡片常显的多行文本框，内容作为 `session.instructions` 下发。

### 7.3 设备 Devices
输入 / 输出两行紧凑展示。

### 7.4 高级设置 Advanced（默认折叠）
收纳 WebSocket URL、采样率、传输方式。

### 7.5 开发者日志 Developer log（默认隐藏）
原 Event log 移入折叠区，默认关闭，带计数徽标；建议仅 dev 模式展开。
