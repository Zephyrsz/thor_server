# Design

## Theme
Dark, polished operator console for realtime voice interactions.

## Color
- Background: deep slate / charcoal
- Surfaces: slightly lifted graphite
- Text: near-white with strong contrast
- Accent: cyan/teal for connection and action
- Status: green for connected, amber for live activity, rose for errors

## Typography
- Use a modern sans serif with neutral proportions.
- Large but restrained headings.
- Body text should stay compact and highly readable.

## Layout
- Full-height app shell with a top status bar.
- Primary column: live transcript + conversation history.
- Secondary column: connection settings, instructions, and controls.
- Mobile stacks into a single column with persistent primary actions.

## Components
- Endpoint field
- Connect / disconnect controls
- Mic start / stop control
- Live transcript panel
- Conversation history stream
- Error / status banners

## Motion
- Minimal and purposeful.
- Use short fades and subtle transitions only.

## Rules
- No decorative marketing framing.
- No oversized card stacks.
- No gradients as the main visual language.
- Keep the page readable and utilitarian.
